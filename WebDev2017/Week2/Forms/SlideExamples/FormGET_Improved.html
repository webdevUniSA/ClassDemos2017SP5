Improved Form Layout Approach

A good form layout approach involves targeting each input element with its own label element.
This is done by giving the input element an id and then pointing the label to the id using the for attribute.
The only issue is what happens with radio buttons where there is more than one input for the label!

<label for="TBFirstName" > First Name: </label >
<input type="text" name="firstname" id="TBFirstName" />
