Hyperlink Examples

Relative URL Links

The current page is in RootFolder/Week1

  • To move from the current folder one folder toward the root, use "../fileName" infront of the linked page
  • To move one folder further into the current folder use "/nextFolderName/fileName" infront of the linked page
  • To return to the root directory use "/fileName" as the link
  • To get the default page of the root directory use "/" as the link
Link to Demo HTML href="../Index.html"
Link to Demo HTML href="/Index.html"
Link to Root Directory href="../"
Link past Root Directory href="../../../"
Quick link to Root Directory href="/"

Absolute URL Links

Absolute URLs specify exactly where on the web a page can be located.
This generally includes the protocol and world wide web: http://www.domain.com
